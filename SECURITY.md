# Security Policy

## Supported Versions

We take the security of the **HomeLab Dashboard** seriously. Below are the versions currently receiving security updates:

| Version      | Supported          |
| -----------  | ------------------ |
| 0.1.0-beta   | ✅ Supported       |
| none         | ❌ Not Supported   |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a potential security vulnerability, please use one of the following methods:

1. **GitHub Private Reporting (Preferred):** Use the [Private Vulnerability Reporting](https://github.com/Vesse00/homelab-dashboard/security/advisories/new) feature on this repository.
2. **Email:** Send a detailed report to **vessepl@gmail.com** with title **[homelab-dashboard]**.

### What to include in your report:
- A description of the vulnerability.
- Steps to reproduce the issue (PoC).
- Potential impact.

## Our Response Process
- We will acknowledge receipt of your report within **48 hours**.
- We will keep you informed of our progress as we work to patch the vulnerability.
- Once fixed, we will publish a Security Advisory and credit you for the discovery (if desired).

Thank you for helping us keep the HomeLab community safe! 🛡️
